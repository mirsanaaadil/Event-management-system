Project Overview

The Event Registration System is a web-based application designed to manage events efficiently and simplify user registrations. It allows admins to create and manage events, while users can view, register, and track their events. The system ensures data consistency and a smooth experience for both administrators and participants.


Key Features:

- User Authentication: Secure login for admins and users with role-based access control.

Admin Capabilities:
- Add, edit, and manage events.
- View registered users for each event.

User Capabilities:
- Browse available events.
- Register for events.
- View a personal list of registered events.

Database Management:
- Stores all users, events, and registrations.
- Maintains relationships between users and events for accurate tracking.

User-Friendly Interface:
- Clean dashboards for both admin and users.
- Tables with hover effects, sorting, and structured layouts.

Session Management:
- Users and admins stay logged in with session tracking.
- Logout functionality for secure session termination.

Technology Stack:

- Backend: Python Flask, SQLite database
- Frontend: HTML, CSS, Jinja2 templates
- Version Control: Git & GitHub


Objective:
The system streamlines event management by automating registrations, reducing manual tracking, and providing an organized and intuitive platform for users and admins.
